# Pagina dei Progetti

## Riguardo la lista di progetti

La lista dei progetti è salvata nel file `projects.yml` all'interno della cartella `_data/projectsInfo`, [qui](https://github.com/PoliNetwork/polinetwork.github.io/blob/master/_documentation/in_data_flags.yml) puoi trovare un file di esempio.

## Riguardo i singoli progetti

Le informazioni riguardanti i singoli progetti si trovano nel file `<nomeprogetto>.md`, che è replicato nelle cartelle `_projectsInfoEN` e `_projectsInfoIT`. L'unica differenza è ovviamente la lingua. [Qui](https://github.com/PoliNetwork/polinetwork.github.io/blob/master/_documentation/in_file_flags.yml) puoi trovare un file di esempio.
