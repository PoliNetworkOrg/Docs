# 🌐 Installing unpacked extensions on Chrome and Edge

## Setting up your browser

- Visit [chrome://extensions](chrome://extensions) (works on Edge too);
- Enable `Developer mode` by flipping the switch on the top right corner (or in the left pane in Edge).

## Installing the extension

- Unpack the `.zip` file you downloaded to a new folder you created. If you open the folder and find some files and an `assets` folder without navigating through other subfolders it means you did right;
- On the [chrome://extensions](chrome://extensions) page, click `Upload unpacked extension` on the top left corner (or `Upload unpacked` right at the top on Edge).

## IMPORTANT footnote

Do not move the folder where you extracted the extention's files. If you do, your browser won't find it anymore and it will appear as though you uninstalled it.
In case you want to move it, you have to repeat the second step of the installation process.
