# Projects page

## About the list of projects

The list of projects is stored in the `projects.yml` file inside the `_data/projectsInfo` folder, [here](https://github.com/PoliNetwork/polinetwork.github.io/blob/master/_documentation/in_data_flags.yml) you can find a sample file.

## About the single project

Information about projects is stored in the `<projectname>.md` file, which is replicated across the `_projectsInfoEN` and `_projectsInfoIT` folders. The only difference is obviously the language. [Here](https://github.com/PoliNetwork/polinetwork.github.io/blob/master/_documentation/in_file_flags.yml) you can find a sample file.
